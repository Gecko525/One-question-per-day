1. 浏览器查找url是否有缓存，并比较缓存是否过期。 2. DNS解析url查找对应的ip地址 //本地hosts => 本地DNS解析器缓存 => 本地DNS解析器 => 迭代查询（域服务器 -> 顶级域 -> 第二层域 -> 子域） 3. 根据ip建立tcp链接（三次握手） //一：客户端发送syn包（syn=j）到服务器，并进入SYN_SENT状态。 => 二: 服务器收到syn,再向客户端发送SYN+ACK包,进入SYN_RECV状态 => 三:客户端收到SYN+ACK包,向服务器发送ack确认包。客户端和服务器进入ESTABLISHED。tcp连接成功。 4. 浏览器向服务器发送HTTP请求 5. 服务器处理请求返回数据给浏览器 （状态码1开头请求已接受；2开头成功接受；3开头重定向；4开头语法错误或请求无法实现；5开头服务的错误） 6 tcp关闭（四次握手） //一：客户端发送FIN请求断开连接 二:服务器发送ACK表示同意 三:服务器发送FIN表示没有数据发送请求断开连接 四：浏览器发送ACK表示同意 7. 浏览器响应并页面渲染
